Sprinkler applet

The applet first build a decision tree based model from a training set and then test it against a test set returning the error rate.

The amount of pre pruning can be set from 0 (no prepruning, full tree generation) to 1 (maximum amount of pre pruning, single node tree).

Record set file format:
An input file starts with comments (each comment line starts with #).
A single special line begging with "# {" and closed by "}" must be present before the record lines. It indicates attribute domains separated by ":", single domain values are separated by comma, as in the following example. Each of the next lines specifies a record where attributes are separated by comma. Last attribute is the label associated with the record.

# {x,o,b:x,o,b:x,o,b:x,o,b:x,o,b:x,o,b:x,o,b:x,o,b:x,o,b:positive,negative} x,x,x,x,o,o,x,o,o,positive x,x,x,x,o,o,o,x,o,positive ...
