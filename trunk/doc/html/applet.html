Sprinkler applet

Abstract:

A demonstrative applet for sprinkler http://code.google.com/p/sprinkler/, an educational decision-tree classifier for categorical data developed in Java.

The applet first build a decision tree based model from a training set and then test it against a test set returning the error rate (the amount of incorrectly classified records on the total number of records).
The amount of pre pruning can be set from 0 (no prepruning, full tree generation) to 1 (maximum amount of pre pruning, single node tree). This value is considered by the termination policy in the tree growing phase and compared with the Gini disparity index of the node.

(A command line interface is provided by sprinkler for more advanced features as iterative validation for Accuracy Vs Purity using .632 validation).

Record set file format:

An input file starts with comments (each comment line starts with #).
A single special line beginning with "# {" and closed by "}" must be present before the record lines. It indicates attribute domains separated by ":", single domain values are separated by comma, as in the following example. Each of the next lines specifies a record where attributes are separated by comma. Last attribute is the label associated with the record. 

      # This is a comment line
      # Next line specifies a record with 9 fields that can assume "x", "o" or "b" values and a last label field
      # {x,o,b:x,o,b:x,o,b:x,o,b:x,o,b:x,o,b:x,o,b:x,o,b:x,o,b:positive,negative}
      x,x,x,x,o,o,x,o,o,positive
      x,x,x,x,o,o,o,x,o,positive
      ...
Datasets:
  • Tic-Tac-Toe training dataset
  • Tic-Tac-Toe test dataset

A few other datasets converted for use for sprinkler and sprinler-applet can be dowloaded from http://code.google.com/p/sprinkler/downloads/list
